\# AsistenteIA Flask + Render



Servidor Flask para análisis de signos vitales y conexión con ESP32.



\## Endpoints

\- `/` → Estado del servidor

\- `/ia` → Procesa solicitudes con IA

\- `/sensores` → Recibe datos de sensores del ESP32



\## Despliegue

\- Plataforma: \[Render.com](https://render.com)

\- Backend: Flask + OpenAI API

\- Frontend: HTML local o página web externa
